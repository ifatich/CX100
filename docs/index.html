Langkah 01 — Instalasi

Ekstrak & Jalankan

Penggunaan bot ini sangat praktis karena instruksi Konfigurasi dan Data Target telah diintegrasikan di dalam alur eksekusi aplikasi.

  1. Ekstrak (unzip) file aplikasi yang telah Anda unduh.
  2. Buka folder dari hasil ekstrak tersebut.
  3. Klik ganda (double-click) pada icon aplikasi CX100 Test.
  4. Ikuti langkah-langkah yang muncul di dalam aplikasi. Aplikasi akan memandu Anda untuk mengatur Konfigurasi dan Data Target.

Langkah 02 — Keamanan

App Password Gmail

Saat aplikasi memandu Anda melakukan konfigurasi, Anda akan dimintai password email. Demi keamanan, Anda wajib menggunakan App Password, bukan password utama email Anda.

Penting Akun Gmail Anda harus sudah mengaktifkan fitur Verifikasi 2 Langkah (2-Step Verification) sebelum bisa membuat App Password.
  1. Kunjungi halaman pengaturan keamanan: App Passwords.
  2. Pada kolom Select app, pilih Other (Custom name) dan beri nama bebas (misal: "CX100 Bot").
  3. Klik tombol Generate.
  4. Simpan 16 karakter password yang muncul di layar untuk dimasukkan ke konfigurasi bot.

Langkah 03 — Panduan Lanjutan

Penjelasan Konfigurasi

Ketika aplikasi berjalan dan mendeteksi konfigurasi masih kosong, ia akan memunculkan peringatan seperti ini:

CX100 TEST — Polling Otomatis
✅ Node.js: v24.6.0
✅ Google Chrome: sudah terinstall
⚠️ Konfigurasi default terdeteksi!
Buka file: app/config.txt
Harap isi GMAIL_OTP dan APPS_PASSWORD dengan benar.
Setelah diisi, tekan ENTER untuk lanjut...

Buka file config.txt (di dalam folder app), lalu sesuaikan isinya:

# Email Gmail untuk menerima kode OTP
GMAIL_OTP=email.anda@gmail.com

# Apps Password Gmail (16 karakter, tanpa spasi)
APPS_PASSWORD=xxxxxxxxxxxxxxxx

# File daftar email (berada di dalam folder yang sama)
FILE_LIST_EMAIL=list_email.example.txt

# Folder untuk menyimpan hasil tangkapan layar (capture)
FOLDER_CAPTURE=capture
  • GMAIL_OTP: Alamat Gmail yang Anda gunakan.
  • APPS_PASSWORD: Password aplikasi sebanyak 16 karakter (didapat dari langkah 02).
  • FILE_LIST_EMAIL: Nama file yang berisi daftar email target.
  • FOLDER_CAPTURE: Path folder lokasi untuk menyimpan hasil capture (disarankan dibiarkan capture agar tersimpan di folder ini).
Langkah 04 — Panduan Lanjutan

Menyiapkan Data Target

Setelah konfigurasi selesai, bot akan mengecek data target Anda. Jika kosong, bot akan memunculkan layar berikut:

CX100 TEST — Polling Otomatis
✅ Node.js: v24.6.0
✅ Google Chrome: sudah terinstall
⚠️ File daftar email kosong!
Buka file: app/list_email.example.txt
Isi dengan daftar email tujuan.
Setelah diisi, tekan ENTER untuk lanjut...

Anda harus mengedit file list_email.example.txt (atau sesuai nama yang Anda atur di konfigurasi) dan memasukkan satu alamat email per baris.

💡 Trik Email Anakan (Dot Trick) Untuk kebutuhan testing massal, Anda cukup menggunakan 1 buah email induk, lalu manfaatkan teknik dot trick untuk menghasilkan variasi (anakan) ke alamat yang sama.

Gunakan bantuan AI (seperti Gemini atau ChatGPT) dengan menyalin prompt berikut:
"Tolong buatkan 100 variasi email anakan menggunakan gaya titik (dot style) dengan akhiran domain @googlemail.com dari email induk nama.anda@gmail.com"
app/list_email.example.txt
# --- CONTOH PENGGUNAAN DOT TRICK ---
# Pastikan menggunakan akhiran @googlemail.com

n.ama.anda@googlemail.com
na.ma.anda@googlemail.com
nam.a.anda@googlemail.com
nama.a.nda@googlemail.com
nama.an.da@googlemail.com
